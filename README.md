# BlakBoks(CaaS)
> Elasticsearch but for compiling untrusted code
Compiler as a Service is a compiler that is available over http/2 and gRPC. 

# Setup
First install the blakboks by :
```bash
go get -u github.com/aitumik/blakboks
```

# Tools
* Golang
* chroot
